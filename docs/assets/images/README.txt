Placeholder Images Needed:
=======================

This folder is for storing images used throughout the website.

Suggested images to add:
- Hero images for each page
- Team member photos (about.html)
- Partner logos (WPI, Soochow University, etc.)
- Project photos (Shilin Narrative Studio gallery)
- Card images for project cards
- Social media icons (or use icon fonts)

For now, the website uses CSS gradient placeholders that will automatically be replaced when images are added.

Image recommendations:
- Format: JPG or PNG
- Size: Optimized for web (under 500KB per image)
- Dimensions: 
  * Hero images: 1920x1080px
  * Card images: 600x400px
  * Team photos: 300x300px (square)
  * Gallery images: 800x800px (square)
